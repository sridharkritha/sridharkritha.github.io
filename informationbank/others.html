Consider this code snippet:

 

#include <stdlib.h>

#include <string.h>

 

char * func1(char * s)

{

            char * t = malloc(strlen(s));

            strcpy(t,s);

            return t;

}

 

1. What language is this code written in?

2. What does this code do?

3. ‘func1’ is not a very good name. What name would you use, and why?

4. Does it compile?  If not, why not?

5. Does it work?  If not, why not?

6. Comment on the robustness.

7. Could this code be any better?  If so, how?

 

 

1.    Code written in C language.

2.    This code will do the string copy operation.

3.    We can give the function name as stringCopy – because it explains the purpose of the function and also increase the readability of the code.

4.    Yes the code will compile if you compile in C compiler but not if you use C++ compiler - gives void* to char* type casting compilation error.

5.    Yes it will work but not always. For example if ‘malloc’ was failed to allocate the requested memory then it will return the NULL pointer. Copying the data to the NULL pointer will leads to crash!.

6.    A. NULL check should be done after the dynamic memory allocation for avoiding the by possible crash due to memory allocation failure.

B. Input argument should be const – for to protect the original data from alteration.

C. Allocate the amount of memory is equal the multiple of source string length and size of char - instead of considering the size of char is 1. Additionally one extra space should be allocated for storing ‘\0’.

D. Allocated memory should be de-allocated by the caller for avoiding the memory leak.

 

7.    Please find below the better function for doing the string copy.

 

#include<stdio.h>

#include <stdlib.h>

#include <string.h>

 

char* stringCopy(const char * s)

{

      char *str = (char*) malloc((strlen(s)*sizeof(char))+1);    

      if(str ==NULL)

      {

            printf("Memory Allocation Failure");

            return NULL;

      }

      else

      {

            strcpy(str,s);

            return str;

      }

}

 

 

int main()

{

      char ary[] = "Hello World";

      char *str = stringCopy (ary);

      printf("%s",str);

      free(str);

      return 0;

}

Loading time required by the plain graphics comparatively more than the loading time required by the HTML page.

HTML files can be easily edited with notepad or WYSIWYG editors .

4.    It is static needs to be manually updated- needs some scripting language support to change it as per the requirements.

            disjoint =0,

                     const int x1,const int y1, const int x'1,const int y'1,)

{

            result temp = disjoint;

             // LHS of Rect A X Axis value is less than RHS of Rect B X Axis Value

            // RHS of Rect A X Axis value is greater than LHS of Rect B X Axis Value

                           ( x'0 > x1 ) &&

            // Top side of Rect A Y Axis value is less than bottom side of Rect B Y Axis Value

                           ( y0  < y'1) &&

            // Bottom side of Rect A Y Axis value is greater than top side of Rect B Y Axis Value

                           ( y'0 > y1 ) );

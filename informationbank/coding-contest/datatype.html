DataType - Limits



For 32-bit and 64-bit compilers, Microsoft Visual C++ recognizes the types shown in the table below. Note that the following type also have unsigned forms:

typedef unsigned char       BYTE;    // 1 Byte
typedef int                 BOOL;    // 4 Byte
typedef unsigned short      WORD;    // 2 Byte = 16 bits (Even for 32bit Processor)
typedef unsigned long       DWORD;   // 4 Byte = 2 x WORD = 32 bits 
typedef unsigned __int64    DWORD64; // 8 Byte = 64 bits
typedef unsigned __int64    QWORD    // 8 Byte = 4 x WORD = 64 bits

  • int (unsigned int)

  • __int8 (unsigned __int8)

  • __int16 (unsigned __int16)

  • __int32 (unsigned __int32)

  • __int64 (unsigned __int64)

  • short (unsigned short)

  • long (unsigned long)

  • long long (unsigned long long)

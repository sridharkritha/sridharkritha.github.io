Set theory is the basic math concept behind the bit manipulation.

Consider a set A = { 2, 4, 5} which can be visualized as a integer with 2nd,4th and 5th bit set.
A = { 2, 4, 5}  => [ 001011]    (range 0 to 5) => 11 (decimal from the binary)
B = { 0 ,3, 4}  =>  [ 100110]    (range 0 to 5) => 38 (decimal from the binary)

A U B = { 0,2,3,4,5}

A || B = 001011
             100110
        -----------
             101111       =>  {0,2,3,4,5}
        -----------

Likewise you can apply any set theory rules.

Some simple "tricks" explained:
(assume lowercase variables are the numbers that represent corresponding sets: i.e. is represented by a).
  1. So I'm talking about sets. Do we have Union? Intersection? Well yeah, if we have , then we have c = a | b. Also, if , then c = a & b. Just check that bit-by-bit the rules of & and | mirror the rules of intersection and union when taken element-by-element
  2. Does set contain element ? This is like asking: Is ? In terms of numbers,                                                                                    "Is ((a & (1<<i)) != 0)"
  3. What is the set {1, 2, 3, ..., n}? Well we need all 1's in the first n bits. We can just take (1<<n) - 1.
  4. How do we remove i from , given we know it is in ? We turn off the 1 at the i'th bit: a - (1<<i).
  5. How do you take a complement of a set when the Universal set is {1, 2, 3, ..., n}? Well, follow a combination of "tricks" 3 and 4 to get:                  ((1 << n) - 1) - a.
  6. What if we don't know whether i is in ? We want ! That's . That's a combination of some earlier rules too!                                (a & ((1 << n) - 1 - (1 << i))). [Also, you can do Balajiganapathi's "Flip ith bit of a" method].

By Balajiganapathi:

// See if ith bit is set in n
(n & (1 << i)) > 0

// Set ith bit of n
n |= (1 << i);

// Reset ith bit of n
n &= ~(1 << i);

// Flip ith bit of n
n ^= (1 << i);

// Get m least significant bits of n
n &= (1 << m) - 1;

// Get the least significant set bit
n & (-n)

#include <iostream>
using namespace std;

// possible subset 0f {1,2,3} => 1, 2, 3, 12, 13, 23, 123
int main()
{
	int N =3;
	int allMasks = (1 << N);
for (int i = 1; i < allMasks; i++)
{
    for (int j = 0; j < N; j++)
        if ((i & (1 << j)) > 0) //The j-th element is used
           cout<<j + 1<<" ";

	cout<<endl;

}

}

Functional Pointer
 
 

Pointers Representations:

    int   *a[10];         // An array of 10 pointers to integers
    int (*a)[10];         // A pointer to an array of 10 integers
    int  (*a[10])(int);   // An array of 10 pointers to functions that take an integer argument and return an integer.

Function Pointer: Function Pointers are used to replace switch/if-statements, to achieve late-binding or to implement callbacks.

One aspect in the case of late-binding is runtime: If you call a virtual function, your program has got to determine which one has got to be called. It does this using a V-Table containing all the possible functions. This costs some time for each call and maybe you can save that time using function pointers instead of virtual functions.

Callback Functions - Application of Functional Pointers:

A callback function is one that is not invoked explicitly. Rather, the responsibility of its invocation is delegated to another function that receives its address. To implement a callback, you need to define an appropriate function pointer first.

Consider the following function prototype:

    void f();

It declares a function called f() that takes no arguments and returns void.

A pointer to such a function has the following type:

    void (*) (); // type of f() - This declaration didn't created a pointer variable yet -- it only declared 
// the type of such a variable. So you can use this type in a typedef declaration, or in a 
// sizeof expression, or as a function parameter:

  Example:

    // get the size of a function pointer
    unsigned psize = sizeof (void (*) ()); 


    // used as a function parameter
    char signal( int data, void (*)() );

Function Pointer declaration: 

    void (*ptr) (); // ptr is a pointer to a function that takes no arguments and returns void.

You can assign a different value to ptr as long as it's the address of a function with the same signature and return type.

For example:

    void fun() { /* do something */ }
    ptr = func;


Passing a Callback Function's Address to Its Caller

    void fun()  { printf("\n This is from fun fuction"); }

    void proc() { printf("\n This is from proc fuction"); }

    void caller(void(*p)()) {
        p(); // calls the bound function
    }

    int main() {
        void (*ptr) ();
        ptr = fun;
        caller(ptr); // pass address of fun to the caller
        ptr = proc;
        caller(ptr); // pass address of proc to the caller
    }


Remember that the assignment to 'ptr' may take place at run time, which enables you to implement dynamic binding.

2.How to Replace a Switch-Statement using Function Pointers ?

     // one of these functions is selected at runtime with a " swicth " or a " function pointer"
    float Plus (float a, float b) { return a + b; }
    float Minus(float a, float b) { return a - b; }


// Solution using switch-statement - <opCode> specifies which operation to execute
    void Switch(float a, float b, char opCode) {
        float result;
        switch(opCode)
        {
            case '+' : result = Plus (a, b); break;
            case '-' : result = Minus (a, b); break;
        }
        printf("Solution is :%f",result ); // display result
    }


// Solution using functional pointer -- <pt2Func> is a function pointer
// Callback function doesn't have input argument vari
    void Function_Pointer(float a, float b, float (*pt2Func)(float, float))
    {
        float result = pt2Func(a, b);      // call using function pointer
        printf("Solution is :%f",result ); // display result
    }

    /* Execute example code */
    void main() {
        printf("Executing function using switch-statement and functional pointer" );
        Switch(2, 5, '+');
        Function_Pointer(2, 5, &Plus); // Note: Callback function doesn't have input argument variable
    }

3. How to Return a Function Pointer ? There are 2 methods to return a function pointer by the " direct " method and by the " typedef " method.

    // METHOD - I
    // Function takes a char and returns a pointer to a function which is taking two floats and returns a float. 
// <opCode> specifies which function to return
    float (*GetPtr1(const char opCode))(float, float) {
        if(opCode == '+')   return &Plus;
        else                return &Minus; // default if invalid operator was passed
    }


    // METHOD - II 
    // Solution using a typedef: Define a pointer to a function which is taking two floats and returns a float
    // typedef float(*pt2Func)(float, float);

    // Function takes a char and returns a function pointer which is defined with the typedef above. 
// <opCode> specifies which function to return 
    pt2Func GetPtr2(const char opCode) {
        if(opCode == '+') return &Plus;
        else              return &Minus; // default if invalid operator was passed
    }

    void main() {
        float (*pt2Function)(float, float) = NULL; // define a function pointer and initialize it to NULL
        pt2Function = GetPtr1('+');
        printf("Solution is %d", (*pt2Function)(2, 4) );  
        pt2Function = GetPtr2('-');
        printf("Solution is %f", pt2Function(2, 4));
    }

4. How to Use Arrays of Function Pointers ?

There are two methods to return a fuction pointer by " direct " method and by " typedef " method. 

    void main() {
        // Method I - using " type-definition" 
        // define arrays and initialize each element to NULL, <funcArr1> arrays with 10 pointers to functions 
// which return an int and take a float and two char 
        typedef int (*pt2Function)(float, float);
        pt2Function funcArr1[10] = {NULL};

        // Method II - Direct method
        // define arrays and initialize each element to NULL, <funcArr2> arrays with 10 pointers to functions 
// which return an int and take a float and two char 
        int (*funcArr2[10])(float, float) = {NULL};

        // Assign the function's address - 'Plus' and 'Minus' 
        funcArr1[0] = funcArr2[1] = &Plus;
        funcArr1[1] = funcArr2[0] = &Minus;

        // Calling a function using an index to address the function pointer 
        printf("%f\n", (*funcArr1[1])(25,16));
        printf("%f\n", (*funcArr1[0])(49,22));
        printf("%f\n", (*funcArr2[1])(56,24 ));
        printf("%f\n", (*funcArr2[0])(34, 19));
    }

Functional Pointer in C++:
 
Function Pointers to Member Functions
Function pointers to member function can be categorized in to two types
1. Pointer to Member Function and
2. Pointer to Static Member Function

1. Static Member Function:  Pointers to static member functions. Since they don't depend on an object instance, they look like ordinary function pointers.

    class myclass {
    public:
        static void myfunc(int x) { std::cout << x << std::endl; }
    };

Now we declare a type for a pointer to the static member function:
    typedef void ( *STATICFUNC ) ( int );

Now assigning a function pointer and calling the function with it is easy:

    void TestStatic() {
        void (*psm)(int) = myclass::myfunc;
        (*psm)( 123 );

        /**** or *****/

        STATICFUNC f = myclass::myfunc;    // assign address of member function. Note: Class name is needed but no need for Object name.
        f ( 123 );                                             // call the function via function pointer
    }

2. Non Static Member Function - Using Typedef:
 Function pointers to non-static member functions. First assume we have again our 'myclass', only that myfunc isn't declared static this time.

The type of a function pointer to it now looks like this:
    typedef void ( myclass::*FUNC ) ( int );

The interesting part of the syntax here is that the asterisk comes after the class name. To use the function pointer, we have to instantiate an object of type myclass, then we can create a function pointer and call the function with it:

    void TestMemberFunc () {
        myclass obj;        // instantiate myclass
        FUNC f = &myclass::myfunc;    // assign address
        ( obj.*f ) ( 123 );    // and call it
    }

The important thing in the syntax here are the parentheses around obj.*f, which basically bind the pointer to the object whose member function we are calling.

2a. Non Static Member Function - Using Template:

In C++, it is good practice use template (T , T&, T*) instead of void (void, void*) data types.

    template <class T>
    void temFun(myclass &obj, T ptr) {  (obj.*ptr)(123); }

    void TestMemberFunc()   {
        myclass obj;  
        temFun(obj, &myclass::myfunc)
    }

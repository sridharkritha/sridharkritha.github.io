UNIX Control-Key Commands

 

Ctrl-c : Interrupt or break character; stops printing and returns to UNIX.

 

Ctrl-h : Erase or backspace a character

 

Ctrl-w : Erase a Word 

 

Ctrl-u :  Clear the command line.

 Ctrl-Alt-d :  Minimize all the Windows.

Redirection

             <          Routes input to command from file

 

            >          Routes output from command to file

 

            >>        Appends output to existing file

 

            |           Routes output between commands

1. npm init
     Creates package.json inside the root folder
2. npm install moment --save
     Creates node_modules folder and Instead of manually downloading the external file (moment.js) from (http://momentjs.com/) - It uses npm install command to download the file. Check the package.json, you will see the downloaded 'moment.js' info under the dependencies.
ex: "dependencies": { "moment": "^2.22.2" }
